🗄️ Conexão Supabase
Banco de dados na nuvem
💡
Preencha as credenciais do Supabase após hospedar. O sistema sincronizará automaticamente todos os dados com o banco na nuvem.
Supabase → Settings → API → Project URL
Chave pública (anon key) — NÃO é a service_role key
⚠️
Importante: No Supabase, vá em Authentication → Policies e desative o RLS (Row Level Security) nas 3 tabelas (chamados, diagnosticos, materiais), ou crie políticas de acesso público. Sem isso, os dados não serão salvos.
SQL para liberar acesso (rode no Supabase)
alter table chamados disable row level security;
alter table diagnosticos disable row level security;
alter table materiais disable row level security;
alter table usuarios disable row level security;
notify pgrst, 'reload schema';
